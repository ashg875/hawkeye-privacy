Hawkeye Chrome Extension Privacy Policy

Effective Date: February 2026

Data Collection

Hawkeye only collects data you explicitly provide in the extension, including:

  • Name
  • Bib number (optional)
  • Photo/selfie (optional)

This information is used solely for organizing and downloading event photos.

Local Storage

All data is stored locally in your browser using Chrome Storage. Hawkeye does not transmit your data to any external servers.

No Other Personal Data Collected

Hawkeye does not collect emails, addresses, passwords, or any other sensitive personal information unless you explicitly provide it in the extension.

User Consent

By using Hawkeye, you consent to the limited local data storage as described above.
