# Hawkeye Chrome Extension Privacy Policy
_Privacy policy for Hawkeye Chrome Extension_  
_Effective Date: February 2026_  

Hawkeye is a Chrome extension designed to help users search, select, and download event photos from publicly available event photography websites.
## 1. Information We Collect
Hawkeye collects only information that the user explicitly provides:
- Name
- Bib number (optional)
- Photo/selfie (optional)

This information is used solely to search for and download event photos selected by the user.

Hawkeye does not collect:
- Email addresses
- Home addresses
- Financial information
- Passwords
- Browsing history
- Location data
## 2. How Data Is Stored
All user-provided information is stored locally on the user's device using Chrome Storage.  
Hawkeye does not transmit, sell, share, or upload any personal data to external servers.
## 3. Website Access & Host Permissions
Hawkeye requires access to specific event photography domains in order to:
- Retrieve publicly available event listings
- Search for photos using user-provided criteria
- Download selected images

The extension only accesses domains necessary for this functionality and only when initiated by the user.
## 4. Downloads Permission
The downloads permission is used solely to allow users to save images to their device.  
No downloaded content is transmitted anywhere else.
## 5. Tabs & Scripting Permissions
The tabs and scripting permissions are used to:
- Open event pages in background tabs
- Retrieve publicly available photo data
- Enable automated downloading

These permissions are not used to monitor browsing activity or collect unrelated data.
## 6. Offscreen Processing
The offscreen permission is used to process images or background tasks without displaying visible browser tabs.
## 7. Data Sharing
Hawkeye does not:
- Sell data
- Share data
- Transfer data to third parties
- Use analytics
- Use tracking technologies

All functionality occurs locally within the user's browser.
## 8. Data Retention
User data remains stored locally until:
- The user deletes it, or
- The extension is uninstalled. Upon uninstallation, Chrome automatically removes stored extension data.
